# 开发中常用的代码段及方法


存储一些开发中经常用的代码片段
* php 文件夹是 php 开发中常用的
* js 文件夹是 js 开发中常用的
* html_css 是一些简单实例


### 目录结构

**php**
```
|-- curl.php    curl以get或post方式请求数据(方法)
|-- myCurl.php  curl以ssl进行发送数据(类)
|-- down_res.php  下载远程文件或图片
|-- str_fwrite.php  将字符串立刻写入日志
|-- count_time_mem.php  统计代码用时及内存情况
|-- log_fwrite.php  自定义到达n行写日志
|-- id_check.php  验证身份证合法性
```
